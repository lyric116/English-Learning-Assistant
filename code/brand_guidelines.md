# 品牌规范（P0-07）

## 品牌定位
- 产品名：英语学习助手（MVP）
- 一句话定位：围绕“输入-理解-测试-反馈”闭环的 AI 英语学习工具。
- 品牌语气：专业、清晰、鼓励行动，不夸张承诺。

## 视觉基调
- 主色：`#2563EB`（可信、学习导向）
- 辅色：`#10B981`（进度与正反馈）
- 强调色：`#F59E0B`（提醒与关注）
- 危险色：`#EF4444`（错误与风险）
- 中性色：`#0F172A`、`#475569`、`#E2E8F0`

## 字体与层级
- 标题：`font-bold`，强调任务目标。
- 正文：中等字重，保证连续阅读可读性。
- 辅助文案：低对比但可读，不使用过度灰化文本。

## 组件文案规范
- 按钮：动词开头，明确结果，例如“生成双语内容”“开始测试”“保存配置”。
- 错误提示：说明原因 + 下一步，例如“Base URL 无效，请检查协议和域名”。
- 成功提示：说明动作结果，例如“已保存到收藏”。

## 交互规范
- 关键动作必须有状态反馈：loading、成功、失败。
- 表单必须有可见标签和可聚焦状态。
- 移动端优先保证点击热区与可读性。

## 禁止项
- 不使用模糊营销语（如“极速提升”“一键精通”）。
- 不输出无法验证的学习效果承诺。
- 不在日志、UI 或分享文本中泄露敏感配置（API key/token）。
