# 目标用户画像（P0-06）

## Persona A：考试冲刺型（考研/四六级/雅思）
- 用户特征：18-26 岁，阶段性备考，时间集中，目标分数明确。
- 核心需求：快速提分、词汇记忆效率、阅读和做题闭环。
- 使用频率：每周 5-7 天，每次 30-60 分钟。
- 成功指标：
  - 周词汇学习量 >= 120
  - 阅读测验平均分 >= 75
  - 错题重练后正确率持续上升

## Persona B：职场提升型（邮件/会议/文档阅读）
- 用户特征：22-35 岁，在职，学习碎片化，偏场景实用。
- 核心需求：高频表达积累、复杂句理解、英文材料阅读能力。
- 使用频率：每周 3-5 天，每次 15-30 分钟。
- 成功指标：
  - 月度阅读篇数 >= 20
  - 句子分析使用次数 >= 40
  - 词汇掌握率（月环比）持续提升

## Persona C：日常长期提升型
- 用户特征：16-40 岁，不以考试为唯一目标，重视长期习惯。
- 核心需求：持续输入、学习反馈、成就感驱动。
- 使用频率：每周 4-6 天，每次 20-40 分钟。
- 成功指标：
  - 连续学习天数 >= 14
  - 周报告建议执行率 >= 60%
  - 阅读-测验-成就链路完成率 >= 70%

## MVP 统一衡量指标
- 7 日留存：用户连续 2 次以上完成“阅读 -> 测验”。
- 学习闭环完成率：一次会话内完成至少 2 个模块。
- AI 功能可用率：AI 请求失败后可恢复率（重试后成功）> 80%。
